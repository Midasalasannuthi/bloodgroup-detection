# Fingerprint-Based Blood Group Detection

A deep learning project that detects blood group from fingerprint images using MobileNet CNN.

## Model
- Algorithm: MobileNet (CNN)
- Accuracy: 91%
- Latency: 1.5 seconds

## Files
- `train.py` - Model training script
- `evaluate.py` - Model evaluation script
- `app.py` - Streamlit web application

## Tech Stack
Python, TensorFlow, MobileNet, Streamlit
